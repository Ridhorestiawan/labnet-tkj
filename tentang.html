Tentang

Apa itu LabNet?

LabNet adalah website dokumentasi implementasi jaringan LAN di laboratorium komputer sekolah. Website ini dibuat sebagai proyek akhir jurusan Teknik Komputer dan Jaringan (TKJ).

Teknologi

Dibangun menggunakan HTML5, CSS3, dan JavaScript murni tanpa framework berat maupun database — ringan, cepat, dan mudah dijalankan di browser manapun.

Fitur

  • Informasi topologi jaringan Star
  • Daftar perangkat jaringan lengkap
  • Monitoring status perangkat secara langsung
  • Informasi kontak pengelola
